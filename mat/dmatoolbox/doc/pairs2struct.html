pairs2struct

PURPOSE

PAIRS2STRUCT Turns field-value pairs into a structure

SYNOPSIS

function s = pairs2struct(varargin)

DESCRIPTION 

PAIRS2STRUCT  Turns field-value pairs into a structure
   S = PAIRS2STRUCT('field1','value1','field2','value2',...), returns
   S.(field1)=value1, etc.
