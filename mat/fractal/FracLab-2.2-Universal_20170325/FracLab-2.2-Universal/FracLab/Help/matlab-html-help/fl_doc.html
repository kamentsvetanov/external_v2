FRACLAB Functions   

fl_doc

Reference page in Help browser

Syntax

fl_doc
fl_doc('fraclab_functionname')
fl_doc fraclab_functionname

Description

fl_doc opens the Help browser displaying the Contents pane.

fl_doc fraclab_functionname displays the reference page for the FracLab function functionname in the Help browser.

Examples

Run fl_doc to display the roadmap page for FracLab (that is, the Fraclab toolbox directory).

Run fl_doc boxdim_classique or fl_doc('boxdim_classique') to display the reference page for the boxdim_classique function in the Fraclab Toolbox.
