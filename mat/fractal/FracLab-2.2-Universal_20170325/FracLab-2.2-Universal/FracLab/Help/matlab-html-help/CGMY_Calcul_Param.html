CGMY_Calcul_Param

Estimates the parameters of a tempered alpha-stable Lévy Motion

Syntax

[Estim_Param]=CGMY_Calcul_Param(sig);

Description

[Estim_Param]=CGMY_Calcul_Param(sig) Assuming sig is a tempered alpha-stable Lévy motion, estimates its parameters. More precisely, the 1D signal sig is assumed to be a CGMY process, and we estimate the parameters alpha (or Y), G, M and Y using a maximum likelihood approach in the Fourier domain [1].

The input sig is any 1D signal. The output Estim_Param is a vector with four components, [C,G,M,Y=alpha]. The estimated parameters appear in the window after computation is completed.

Example : synthesis of a tempered Lévy motion and estimation

    N=16000; alpha= 1.5; C=0.001;G=2;M=2;
    sig = CGMY_Principal(alpha,C,G,M,N,NaN);
    [Estim_Param3]=CGMY_Calcul_Param(sig)
