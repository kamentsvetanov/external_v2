flvariaIR

Computes a Increments Ratio Statistics based estimation of the Holder exponent for a 1D signal

Syntax

[HIR]=flVariaIR(sig,winsize)

Description

[HIR]=flVariaIR(sig,winsize)Estimates the Holder function H of the input signal sig using increments ratio statistics. The paramater winsize characterizes the neighborhood of each point used for the computation.

The size of this neighborhood is equal to $n^{1-winsize}$, where n is the number of points in the signal. The neighborhood is centered around the current point.



Examples

    N = 2048; t = linspace(0,1,N); Ht=eval('0.5+0.3*sin(4*pi*t)');
    x = mBmQuantifKrigeage(N,Ht,10);
    Hir = flVariaIR(x,0.35);
    figure; plot(Hir,'k')
    xlabel('number of points');ylabel('Holder Exponent');
