4. Create graph

This allows to build a structure of type graph. Enter a Name for your graph or Get it from the Variables list (recall however that this is an output name, thus if you get a name, your original data with this name will be erased). Enter the name of the vector containing the abscissa on the line Data1, and the name of the vector containing the ordinates on the line Data2 (or get them). You may also give a Title, X label and Y label. When you're done, hit Create.
