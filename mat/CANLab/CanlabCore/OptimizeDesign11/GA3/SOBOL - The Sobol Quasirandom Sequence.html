SOBOL - The Sobol Quasirandom Sequence

SOBOL is a MATLAB implementation of the Sobol quasirandom sequence.

A quasirandom or low discrepancy sequence, such as the Faure, Halton, Hammersley, Niederreiter or Sobol sequences, is "less random" than a pseudorandom number sequence, but more useful for such tasks as approximation of integrals in higher dimensions, and in global optimization. This is because low discrepancy sequences tend to sample space "more uniformly" than random numbers. Algorithms that use such sequences may have superior convergence.

SOBOL is an adapation of the INSOBL and GOSOBL routines in ACM TOMS Algorithm 647. The original code can only compute the "next" element of the sequence. The revised code allows the user to specify the index of the desired element.

The original, true, correct version of ACM TOMS Algorithm 647 is available in the TOMS subdirectory of the NETLIB web site. The version displayed here has been converted to FORTRAN 90, and other internal changes have been made to suit me.

SOBOL is also available in a C++ version and a FORTRAN90 version.

Source code files you may copy include.

  • bit_hi1_base_2.m, returns the position of the high 1 bit base 2 in an integer.
  • bit_lo0_base_2.m, returns the position of the low 0 bit base 2 in an integer.
  • exor.m, computes the bitwise exclusive OR of two integers.
  • get_seed.m, returns a seed for random number generation.
  • i_random.m, returns a random integer in a given range.
  • r_random.m, returns a random real number in a given range.
  • sobol.m, generates a new quasirandom Sobol vector with each call;
  • timestamp.m, prints the current YMDHMS date as a time stamp;
  • uniform.m, returns a random real number in [0,1].

Test files you may copy include:

  • sobol_test.m, runs all the tests;
  • sobol_test01.m, tests exor;
  • sobol_test02.m, tests bit_hi1_base_2;
  • sobol_test03.m, tests bit_lo0_base_2;
  • sobol_test04.m, tests sobol;
  • sobol_test05.m, tests sobol, showing how the value of the seed can be manipulated;
  • sobol_test.out, output from a run of the tests;
